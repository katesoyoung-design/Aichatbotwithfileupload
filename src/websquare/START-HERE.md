## 🎯 이 시스템으로 할 수 있는 것

### ✨ 3가지 모달 모드
1. **플로팅 창** (버튼 1 - 보라색)
   - 드래그로 위치 이동 가능
   - 작고 가벼운 채팅창

2. **폼과 함께** (버튼 2 - 분홍색)
   - 왼쪽: 개인정보 입력 폼
   - 오른쪽: AI 챗봇
   - 채팅으로 입력하면 폼에 자동 입력!

3. **전체화면** (버튼 3 - 파란색)
   - 전체 화면 활용
   - 집중 작업에 최적화

### 🎨 3가지 테마
- 블루퍼플 (기본)
- 미래에셋 AI
- 밤우주

### 🤖 자동 폼 입력
채팅창에 입력하면 자동으로 폼에 입력됩니다:
```
이름: 홍길동
010-1234-5678
hong@example.com
주소: 서울시 강남구 테헤란로 123
```

### 📎 파일 업로드
- 이미지, PDF, 문서 등 업로드 가능
- 업로드한 파일 내용 분석 (시뮬레이션)
- 자동으로 폼에 입력

## 🎬 시작하는 3가지 방법

### 방법 1: 빠른 데모 (추천! ⭐)

```xml

<script src="/websquare/js/chatbot-widget.js"></script>
```

```

```
